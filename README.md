# 3D-RPG

Starter scaffold for a browser 3D dungeon-runner RPG using three.js + TypeScript.

Run locally:

1. npm install
2. npm run dev
3. Open http://localhost:5173

Notes:
- Place any GLB models under public/assets/ (e.g. public/assets/hero.glb)
- This repo is a starter: iterate on systems (AI, combat, physics, animations) as needed.
